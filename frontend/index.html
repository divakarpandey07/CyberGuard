📡 Telegram AlertsFREE
Setup (2 minutes):
1. Open Telegram → search @BotFather
2. Send /newbot → follow steps → copy token
3. Send any message to your bot
4. Visit api.telegram.org/bot<TOKEN>/getUpdates
5. Copy chat_id from response
6. In config.py: set TELEGRAM_ENABLED=True, paste token & chat_id
📧 Email AlertsFREE
Gmail Setup:
1. Google Account → Security → 2-Step Verification → App Passwords
2. Generate password for "Mail" → copy 16-char code
3. In config.py: set EMAIL_ENABLED=True
4. Set EMAIL_SENDER, EMAIL_PASSWORD, EMAIL_RECIPIENT
🔒 Threat IntelligenceFREE tier
AbuseIPDB Setup (optional):
1. Register free at abuseipdb.com
2. Go to Account → API → Create Key
3. Free tier: 1000 checks/day
4. In config.py: set ABUSEIPDB_API_KEY="your-key"

✅ Works without key (just skips IP reputation)
